## 🚀 About This Project

This is my **first real frontend project** — a pixel-as-close-as-I-could-get clone of the Starbucks homepage. No frameworks, no shortcuts. Just HTML, CSS, and a bit of JavaScript.

I followed a tutorial line by line, but made sure I understood every single decision before moving on. When something broke, I debugged it. When something didn't make sense, I stopped and looked it up.

It took me **roughly 2 days** and honestly it was a grind — but seeing it come together in the browser made it worth it.

---

## 🧠 What I Learned

- **HTML5 semantics** — proper use of `<nav>`, `<section>`, `<footer>` and meaningful structure
- **CSS Custom Properties** — using `:root` variables for colors and reusable values
- **Flexbox** — navbar layout, centering elements, aligning items
- **CSS Grid** — two-column layouts with `grid-template-columns` and `gap`
- **Responsive Design** — media queries for mobile, tablet and desktop breakpoints
- **Hamburger Menu** — toggle animation with CSS transforms and JS classList
- **CSS transitions & transforms** — smooth animations for the mobile menu
- **Git & GitHub** — first time pushing a real project to a repo 🎉

---

## 😤 What Was Hard

- Getting the **grid-reversed layout** to work correctly on mobile (order property)
- **Centering things** — turns out `align-items` and `justify-content` are not the same thing and I learned that the hard way
- The **media query nesting bug** — accidentally put one media query inside another and spent way too long figuring out why nothing was working
- Typos in class names that broke entire sections silently (looking at you, `.hamburguer` 👀)
- Staying motivated at 23:00 when everything looked slightly off but you're too tired to care
